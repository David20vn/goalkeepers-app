# Goalkeepers App

Plataforma marketplace para la contratación de arqueros en partidos de fútbol amateur.

Permite la interacción entre jugadores y arqueros mediante un sistema de ofertas bidireccionales con precio fijo, facilitando la publicación de partidos, el envío de ofertas y la asignación de arqueros.

## Funcionalidades principales

- Registro y autenticación de usuarios por rol (jugador / arquero)
- Publicación y gestión de partidos
- Sistema de ofertas entre jugadores y arqueros
- Asignación automática de arquero al aceptar una oferta
- Calificaciones de arqueros por partido
